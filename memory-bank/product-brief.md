# Product Brief — 端口扫描器

## 项目目标
开发一个跨平台端口扫描器，用于课程设计"设计题目四：端口扫描"的提交。

## 核心需求
1. **网段扫描** — 输入 IP 范围（如 192.168.1.0/24），自动发现存活主机
2. **端口扫描** — 使用 TCP connect 方式，扫描指定主机的端口范围（如 1-1024）
3. **主机发现** — 使用 ICMP echo 扫描，检测局域网内存活主机
4. **结果展示** — 清晰展示"主机IP → 开放端口列表"
5. **结果保存** — 支持导出为 JSON / CSV 文件

## 用户角色
- 课程设计答辩老师（最终评委）
- 开发者本人（演示 & 调试）

## 非功能性需求
- 跨平台：Windows 和 Linux 均可运行
- 零外部依赖：仅使用 Python 标准库
- 模块化：扫描引擎与 UI 分离，可 CLI 可 GUI
- 多线程：并行扫描提升速度
